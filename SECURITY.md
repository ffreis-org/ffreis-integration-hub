# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue to report a security vulnerability.

Instead, use GitHub's private advisory feature to report it confidentially:

**[Report a vulnerability](https://github.com/FelipeFuhr/ffreis-integration-hub/security/advisories/new)**

You will receive a response as soon as possible. If the vulnerability is confirmed,
a fix will be prepared and a coordinated disclosure will be made.
